Introduction¶

Python is a powerful high-level, object-oriented programming language created by Guido van Rossum. It has simple easy-to-use syntax, making it the perfect language for someone trying to learn computer programming for the first time. Python is popular for it's extremely large number of library which integrates into any platform

This tutorial is for beginners who are looking to work in python rather study theories. This will cover all the necessary syntax and function, neither more nor less. There are two versions of Python - 2 and 3. This tutorial will be covering syntaxes in Python 3 because its new and more reliable. Python 3 is recommended, however if you are not sure, go through the differences and make sure it meets your requirements. This tutorial will be easier if you are famalier with any other language like C/C++, Java etc.

Indentation and IF...ELSE Statement¶

Python brings a totally different concepts for blocks of code. In most of the language curly braces - { } are used for blocks of code but in Python the block of code must be indented from the left margin.

C++ and other language¶

Main block line 1
main block line 2
{
     Sub block line 1
     Sub block line 2
     Sub block line 3
}
Main block line 3

Python¶

Main block line 1
main block line 2:
     Sub block line 1
     Sub block line 2
     Sub block line 3
Main block line 3

Note¶

  • TAB and 4 spaces are not same when using indentation.
  • There is no restriction on the spaces but two/four spaces or TAB is recommended.
  • Note the colon " : " before Sub block of python. That is the syntax for most statement which require sub block like IF...ELSE and Loops. You will get famalier with this in the next section
    • </ul> As a beginner it might take some time to get comfortable with identation. When indentation error prompts repeatedly, check for TAB and space inconsistency.

IF...ELSE¶

# Intialising a variable

# If...else statement

    print(num, "is a even number.")
else:
    print(num, "is a odd number.")
print("This is always printed.")

3 is a odd number.
This is always printed.

  • The expression in If statement doesn't have braces ( ) - like other languages.
  • Note the colon : after If and Else statement.
