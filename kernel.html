GAN to generate apple or pencil according to the feeded data¶

This GAN model is trained to generate apple or pencil image according to the input data given by user.

Since the processing of colored images takes a lot of time, this model is trained using Hand drawn figure dataset from Google Draw. This is a basic version which will be extended further to colorful original images.

Generator¶

The following generator model is similar to CNN. It takes two input : latent_input | size - 100 | takes random values label_input | size - 1 | takes value {0, 1}

The output of this model is a 28x28 matrix which actually corresponds to the generated images whose dimension are 28x28 pixels.

GAN model¶

There are three models.

  1. Generator
  2. Dicriminator
  3. GAN - It is actually [Generator + Discriminator] appended model

Note¶

While training this GAN, these are the steps that we follow.

  1. Make the discriminator trainable (If we make trainable = False, the weights gets fixed)
  2. Train the discriminator using the actual datasets(y as corresponding label) and the generated image from generator (y as some different label corresponding to fake image)
  3. Make the discriminator non-trainable. Now the weights in discriminator are fixed.( Note that this discriminator is also appended in GAN model)
  4. Train the GAN with latent_input and label_input(described above) corresponding the label_input as y. This forces the generator to generate images which gives corresponding y as output by modifying the weights.

def random_mnist_set(X, y, size=2):
    length = len(X)
    indices = np.random.choice(length, size)
    return X[indices], y[indices]

def random_generated_set(generator, size=2):
    latents = np.random.normal(0, 1, (size, 100))
    return generator.predict_on_batch({'latent_input':latents,
                      'label_input':np.random.randint(2, size=size)})

MAIN TRAINING CODE¶

In [15]:
save_interval = 300
epochs = save_interval*10+1
batch_size=20
history = {'dloss':[], 'gloss':[]}
for iteration in range(epochs):
    discriminator.trainable=True
    X_sample, y_sample = random_mnist_set(x, y, batch_size)
    dloss_real = discriminator.train_on_batch(X_sample, y_sample)
    dloss_fake = discriminator.train_on_batch(random_generated_set(generator, size=batch_size), np.full(batch_size, 2))
    
    discriminator.trainable=False
    
    gan_labels = np.random.randint(2, size=batch_size)
    gloss = gan.train_on_batch({'latent_input':np.random.normal(0, 1, (batch_size, 100)),
                        'label_input':gan_labels}, 
                       gan_labels)
    
    history['dloss'].append((dloss_real[0]+dloss_fake[0])/2)
    history['gloss'].append(gloss[0])
    if iteration % save_interval == 0:
        print('generator: loss={}. acc={}'.format(gloss[0], gloss[1]))
        print('discriminator: loss_real={}, acc_real={}'.format(dloss_real[0],dloss_real[1]))
        print('    loss_fake={}, acc_fake={}'.format(dloss_fake[0],dloss_fake[1]))
        generator.save_weights('generator_{0:05d}.h5'.format(iteration))
